# moex_portfolio
# 📈 Оптимизация инвестиционного портфеля по Марковицу

Проект по анализу недельных котировок акций Московской биржи и построению портфеля минимального риска. 
Цель проекта — загрузить исторические данные по акциям, рассчитать недельные доходности, построить ковариационную матрицу и найти идеальные веса портфеля, минимизирующие историческую дисперсию.

---

## 1. Данные

В проекте используются CSV-файлы с историческими данными котировок. Каждый файл соответствует одной акции. Например:
* `YNDX_W1.csv`
* `SBER_W1.csv`
* `GAZA_W1.csv`

Для расчётов используются цены закрытия (Close), так как доходности считаются между ценами закрытия соседних периодов. Исходные файлы должны лежать в директории `data/`.

---

## 2. Методика расчёта

Для каждой акции берётся последовательность цен закрытия. Процентная доходность рассчитывается по формуле:

$$r_t = \frac{P_t}{P_{t-1}} - 1$$

где:
* $P_t$ — цена текущего периода;
* $P_{t-1}$ — цена предыдущего периода.

После этого строится ковариационная матрица доходностей. Портфельный риск считается по формуле:

$$w^T \Sigma w$$

где:
* $w$ — вектор весов акций в портфеле;
* $\Sigma$ — ковариационная матрица доходностей.

Оптимизатор решает задачу минимизации риска (через библиотеку `scipy.optimize`):

$$\min w^T \Sigma w$$

при строгих ограничениях:
* $\sum w_i = 1$ (весь капитал распределяется, сумма долей равна 100%)
* $0 \le w_i \le 1$ (короткие позиции запрещены)
